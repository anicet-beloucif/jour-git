Commandes Git

Commande Git Utilité
Git config Configurer les préférences de l'utilisateur
Git init Créer un nouveau dépôt Git
Git add Ajouter des fichiers à l'index
Clone Git Utilisé pour la vérification des dépôts. Clone le repo en local
Git commit Valider les modifications apportées
Git status Liste les fichiers modifiés ainsi que les fichiers qui doivent encore être ajoutés ou validés
Git push Envoie les modifications locales apportées à la branche principale associéee
Git checkout Créer des branches ou basculer entre elles (changer le chemin)
Git remote Se connecter à un dépôt distant
Git branch Répertorier, créer ou supprimer des branches
Git pull Fusionner les modifications entre le repo et le stockage local
Git merge Fusionner une branche dans la branche active
Git diff Lister les conflits
Git tag Attribuer des fichiers à une commande "Commit" spécifique
Git log Génerer le log d'une branche
Git reset Réiitialiser l'indexet le répertoire de travail à l
Git rm Supprimer des fichiers de l'index et du répértoire de travail
Git stash Enregistrer des changements sans pour autant les commit immédiatement. "Commit temporaire"
Git show Afficher des informations sur un fichier Git
Git fetch Extraire tout les fichiers d'un reo manquant en local. Uniformiser le local avec le repo.
Git ls-tree Afficher un fichier arborescent, + valeur SHA-1
Git cat-file Afficher le type d'un fichier en se basant sur sa valeur SHA-1
Git grep Rechercher un terme
Gitk Appeler l'interface graphique du dépôt local
Git instaweb Afficher un serveur web du dépôt local
Git gc Optimiser le dépôt en supprimant les fichiers inutiles
Git archive Archiver (.tar/.zip) un arbre
Git prune Suppression des fichiers sans pointeurs entrants
Git fsck Vérifier l'intégrité d'un dépôt Git
Git rebase Réappliquer un commit sur une autre branche
